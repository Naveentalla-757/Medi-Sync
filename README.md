# 🏥 MediSync – Hospital Management System

## 📌 Overview

**MediSync** is a full-stack Hospital Management System designed to streamline healthcare operations. It provides a centralized platform for managing patients, doctors, appointments, and administrative activities efficiently.

The system includes an **Admin Dashboard** that gives complete control and insights into hospital workflows, including patient records, scheduling, and performance statistics.

---

## 🚀 Features

### 🔐 Authentication

* Secure login & registration system
* Role-based access (Admin, Doctor, Patient)

### 📊 Admin Dashboard

* View and manage:

  * Patients
  * Doctors
  * Appointments
  * Leave requests
* Monitor daily hospital activities
* Access real-time statistics and reports
* Manage system settings and logout functionality

### 👨‍⚕️ Doctor Module

* View assigned appointments
* Provide feedback and medical notes
* Manage availability

### 🧑‍🤝‍🧑 Patient Module

* Register and login
* Book appointments with doctors
* View appointment history
* Receive feedback and prescriptions

### 📅 Appointment Management

* Easy appointment booking
* Schedule tracking (day-to-day)
* Status updates (pending/approved/completed)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/MediSync.git
cd MediSync
```

### 2️⃣ Backend Setup (Spring Boot)

* Open backend folder in IDE (IntelliJ / Eclipse)
* Configure MySQL in `application.properties`
* Run the application

### 3️⃣ Frontend Setup (React)

```
cd frontend
npm install
npm run dev
```
